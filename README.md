# SwiftCart Ecommerce
A dynamic e-commerce website is built by using:
- Vanilla JavaScript
- Tailwind CSS
- FakeStore API

# Features
- Dynamic category loading
- Product filtering by category
- Responsive design
- Loading indicator
- Clean UI layout
